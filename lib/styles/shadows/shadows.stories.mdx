## How to use

You can use with a Card component or from the theme.

```tsx
<Card elevation={2}>Shadow</Card>
```

```tsx
<Box
  sx={{
    boxShadow: (theme) => theme.shadows[2]
  }}
>
  Shadow
</Box>
```
